My Game Rankings

I have played many games inlcuding both board/card games and video games. For the purpose of the word game board and card games include in person games as well.

  • 10: Clash Royale

    Clash Royale is a great game for mobile players. A game that balances the in game card collecting feature and the tactical strategic gameplay. However the game's popularity has been decreasing making the match making and pairing not consistant.

  • 9: Castaway

    Castaway is a browser game where the main character who you play as has lost his memory. You travel through multiple islands collecting gems and can acquire various pets.

  • 8: Arcuz

    This game like Castaway is a browser game. This game is more about the adventure and has impressive combat mechanics.

  • 7: Sock Wrestling

    This is a two player game where each player wears only one sock. Each player has one objective and that is to pull the other player's sock off.

  • 6: Kick the Can

    This is an incredibly fun game where one player is "it". They pick an object to guard and all of the other players try to reach the object.

  • 5: Uno

    Uno is a classic card game where each player's goal is to get rid of their hand. First player to do this is the winner.

  • 4: Wizard101

    This game although geared toward a younger audience is still a lot of fun. The game takes shape by using spells in the form of cards. You play a wizard who saves multiple worlds.

  • 3: Starcraft

    This is an older game that really created professional gaming in the first place. The game is incredibly strategic where both characters control an army and try to defeat the other.

  • 2: High-Low

    This is a card game where each player has half the deck and the goal is to run out of cards first by playing them in ascending or descending order to the two stacks in the middle.

  • 1: Minecraft

    What is there to say about this game? It is incredible. There is something in this game for everyone. It has limitless possibilities. You can build, battle, create, play, and do anything.
